# Customer Churn Prediction & Analysis

## Project Overview

This project predicts customer churn using Machine Learning and visualizes customer behavior using Power BI.

The goal is to identify customers who are likely to leave the telecom service and provide business insights to improve customer retention.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Power BI
* Joblib

---

## Dataset

Telco Customer Churn Dataset

Total Customers: 7043

Features: 21

---

## Machine Learning Model

Algorithm Used:

* Random Forest Classifier

Model Accuracy:

* 79.49%

---

## Key Business Insights

* Churn Rate is 26.54%.
* Month-to-Month contract customers have the highest churn rate.
* Fiber Optic users show higher churn compared to DSL users.
* Electronic Check customers are more likely to churn.
* Customers with Two-Year contracts have the lowest churn rate.
